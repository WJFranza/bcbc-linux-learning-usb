# BCBC Linux Learning USB

Explore Linux safely. Learn by doing. Break nothing.

A bootable Linux learning system built around Ventoy and a custom BCBC menu interface.

---

## 🚀 What this does

This project allows you to:

- Run multiple Linux distributions without installing
- Compare GNOME, KDE, and XFCE desktop environments
- Learn Linux safely with zero risk
- Use AI-assisted prompts for guidance

**Nothing on your system is modified unless you choose to install.**

---

## 🧠 Why this exists

Most people want to try Linux but are afraid of breaking their system.

This removes that barrier completely.

Boot from USB → Explore → Reboot → Done

---

## 🖥️ Included Learning Environments

- Ubuntu (GNOME)
- Kubuntu (KDE)
- Xubuntu (XFCE)
- Debian Live
- Zorin OS Core
- Garuda KDE

---

## ⚙️ Setup

1. Install Ventoy to a USB drive  
2. Download your preferred Linux ISOs  
3. Copy ISOs to the USB  
4. Copy the BCBC menu files into `/BCBC/`  
5. Open `README_FIRST.html` after boot  

---

## 🤖 AI-Assisted Help

This project includes prompts you can use with ChatGPT to:

- choose the right distro  
- understand desktop environments  
- get beginner-friendly explanations
